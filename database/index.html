DDL dan DML

Data Definition Language (DDL) dan Data Manipulation Language (DML) adalah bagian dari DBMS (Database Management System) yang berisi perintah-perintah untuk membuat, merubah maupun memodifikasi dan menghapus data pada database.

Let's Start

Structured Query Language

SQL adalah suatu bahasa khusus yang digunakan untuk membuat dan mengolah database.

Data Definition Language

DDL adalah sub perintah dari bahasa SQL yang digunakan untuk membangun kerangka sebuah database, dalam hal ini database dan table.

Data Manipulation Language

DML adalah kumpulan perintah SQL yang berhubungan dengan pekerjaan mengolah data di dalam table.

Database Management system

DBMS adalah sistem pengorganisasian dan sistem pengolahan Database pada komputer.

Data Definition Language

Data Definition Language (DDL) adalah kumpulan perintah SQL yang dapat digunakan untuk membuat dan mengubah struktur dan definisi tipe data dari objek-objek database seperti tabel, index, trigger, view, dan lain-lain.

Perintah dalam DDL

CREATE

Digunakan untuk membuat basis data maupun objek-objek basis data.
Membuat Database:
CREATE DATABASE nama_database
Membuat Tabel:
CREATE TABLE nama_tabel ( nama_kolom1 tipe_data, nama_kolom2 tipe_data )

ALTER

Digunakan untuk menambah, menghapus, dan modifikasi kolom dalam tabel.
Menambah kolom dalam tabel:
ALTER TABLE nama_tabel ADD nama_kolom tipe_data
Menghapus kolom dalam Tabel
ALTER TABLE nama_tabel DROP COLUMN nama_kolom
Mengubah tipe data dari kolom didalam Tabel
ALTER TABLE nama_tabel
ALTER COLUMN nama_kolom tipe_data

DROP

Digunakan untuk menghapus Database, Tabel, Index.
Menghapus Database:
DROP DATABASE nama_database
Menghapus Tabel:
DROP TABLE nama_tabel
Menghapus Index:
ALTER TABLE nama_tabel DROP INDEX nama_index

Data Manipulation Language

DML adalah sekumpulan perintah yang di gunakan untuk melakukan pengelolaan database seperti membuat baru, menambah, menghapus dan menampilkan, perintah utamanya adalah seperti UPDATE, DELETE, INSERT, SELECT.


Perintah dalam DML

SELECT

Digunakan untuk menampilkan atau memilih isi dari sebuah atau lebih tabel.
SELECT nama_kolom1, nama_kolom2 FROM nama_tabel

INSERT

Digunakan untuk memasukkan atau menyisipkan data ke dalam baris dalam Tabel.
INSERT INTO nama_table VALUES (nilai_field1, nilai_field2);

UPDATE

Digunakan untuk mengubah data
UPDATE nama_tabel SET nama_field = nilai, nama_field = nilai WHERE nama_field = nilai

DELETE

Digunakan untuk menghapus data dalam Tabel.
DELETE FROM nama_tabel WHERE nama_field = nilai
